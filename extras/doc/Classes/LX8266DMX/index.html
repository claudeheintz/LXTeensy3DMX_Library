LX8266DMX

Declared In:
LXESP8266UARTDMX.h

Introduction

LX8266DMX is a driver for sending or receiving DMX using an ESP8266 UART1 which has TX mapped to GPIO2 is used for output UART0 which is mapped to the RX pin is used for input

LX8266DMX output mode continuously sends DMX once its interrupts have been enabled using startOutput(). Use setSlot() to set the level value for a particular DMX dimmer/address/channel.

LX8266DMX input mode receives DMX using the ESP8266's UART0 RX pin LX8266DMX continuously updates its DMX buffer once its interrupts have been enabled using startInput() and DMX data is received by UART0. Use getSlot() to read the level value for a particular DMX dimmer/address/channel.

Discussion

Sets a pointer to a function that is called on the break after a DMX frame has been received. Whatever happens in this function should be quick! Best used to set a flag that is polled outside of ISR for available data.

setMaxSlots

Sets the number of slots (aka addresses or channels) sent per DMX frame. 

public

void setMaxSlots (
    int slot);
Parameters
slot

the highest slot number (~24 to 512)

Discussion

defaults to 512 or DMX_MAX_SLOTS and should be no less DMX_MIN_SLOTS slots. The DMX standard specifies min break to break time no less than 1024 usecs. At 44 usecs per slot ~= 24
