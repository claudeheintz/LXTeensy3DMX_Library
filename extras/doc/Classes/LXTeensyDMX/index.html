LXTeensyDMX

Declared In:

Introduction

LXTeensyDMX is a driver for sending or receiving DMX using a Teensy 3.1/3.2's UART0 RX pin 0, TX pin 1

LXTeensyDMX output mode continuously sends DMX once its interrupts have been enabled using startOutput(). Use setSlot() to set the level value for a particular DMX dimmer/address/channel.

LXTeensyDMX input mode continuously receives DMX once its interrupts have been enabled using startInput() Use getSlot() to read the level value for a particular DMX dimmer/address/channel.

LXTeensyDMX is used with a single instance called Teensy3DMX .

Discussion

Sets a pointer to a function that is called on the break after a DMX frame has been received. Whatever happens in this function should be quick! Best used to set a flag that is polled outside of ISR for available data.

public

    int slot);
Parameters
slot

the highest slot number (~24 to 512)

Discussion

defaults to 512 or DMX_MAX_SLOTS and should be no less DMX_MIN_SLOTS slots. The DMX standard specifies min break to break time no less than 1024 usecs. At 44 usecs per slot ~= 24
